## 📖 Project Overview

This repository presents a **Machine Learning-based Network Intrusion Detection System** developed using the **UNSW-NB15** cybersecurity dataset.

The objective of this project is to detect malicious network traffic by comparing classical machine learning algorithms with deep learning techniques for anomaly detection.

The project evaluates different **training strategies**, **feature selection methods**, and **hyperparameter optimization techniques** to improve detection performance.

---

# 🎯 Objectives

- Detect anomalous network traffic
- Compare multiple anomaly detection algorithms
- Evaluate Benign-only vs Contaminated training
- Improve detection using Feature Selection
- Optimize models using Optuna
- Compare model performance on different dataset sizes
